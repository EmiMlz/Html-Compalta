Diapositiva 6

IFrames

Se utiliza para insertar un sitio o página dentro de otra página. Ejemplos: videos de youtube, tweets, etc...

La etiqueta es 'iframe', el atributo 'src' es obligatorio, ya que hace referencia a lo que va a mostrar el iframe, atributos 'width' y 'heigth' para modificar el tamaño, atributo 'frameborder' para quitar o poner bordes, atributo 'SCROLLING' con valores 'yes'(Muestra siempre la barra de scroll)y 'no'(Nunca se muestra el scroll), de todas formas con el valor por defecto funciona bien asi que no hace falta muchas veces usar este atributo.

Etiqueta 'audio' igual que iframe pero para un archivo de audio

Etiqueta 'video' igual que iframe pero para un archivo de video

Atributos para ambos 'video' y 'audio'

  • autoplay: reproducir automáticamente al cargar la pagina. No funciona por políticas de los navegadores modernos.
  • loop: reproducir en bucle

Atributo solo para 'audio'

  • preload: precargar el archivo de audio en segundo plano por el navegador, posibles valores: none, auto
  • controls: hace que el navegador nos provea de una interfaz con controles para la reproducción y el control del volumen

Atributos solo para 'video'

  • poster: Ubicación de una imagen para mostrar cuando no se está reproduciendo el archivo.
  • muted: la ejecución inicia silenciada
  • controls: muestra los botones play / stop / pause
  • width / height
